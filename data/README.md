# data/

Persistent data storage — grows organically as features are used.

| Subdirectory | Contents |
|---|---|
| `executions/` | Trade execution records |
| `market/` | Market data dumps, price history |
| `snapshots/` | Portfolio snapshots (JSONL) |
| `research/` | Research data, analysis results |

SQLite database (`hq_blockchain.db`) is also stored here.
