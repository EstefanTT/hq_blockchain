# testing/

Test files — mirrors the source structure for easy navigation.

| Subdirectory | Tests for |
|---|---|
| `connectors/` | Chain, DEX, CEX, API adapter tests |
| `core/` | Execution, market, portfolio, risk logic tests |
| `strategies/` | Strategy tick, init, shutdown tests |
| `api/` | API endpoint tests |
| `fixtures/` | Shared test data (mock prices, fake orders, etc.) |
| `utils/` | Test helper utilities |
