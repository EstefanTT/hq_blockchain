# utils/

General-purpose utility functions — shared across all layers. No business logic.

| File | Purpose |
|---|---|
| `sleep.js` | Promise-based delay |
| `math.js` | Precision math for financial calculations |
| `formatting.js` | Number/string formatting (currency, percentage, address truncation) |
| `time.js` | Timestamp helpers, duration formatting, UTC date handling |
| `ids.js` | Unique ID generation (for orders, executions, strategy instances) |
